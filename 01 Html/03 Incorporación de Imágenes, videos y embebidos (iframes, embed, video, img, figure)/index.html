Incorporación de Imágenes, Videos y Contenidos Embebidos

1. Imágenes

Las imágenes se insertan usando la etiqueta <img>. Aquí están los atributos esenciales y cómo usarlos:

Ejemplo de Imagen

Atributos importantes:

  • src: Ruta de la imagen.
  • alt: Texto alternativo para accesibilidad y SEO.
  • width y height: Controlan el tamaño de la imagen.
  • title: Texto que aparece al pasar el ratón.

Buenas prácticas:

  • Optimizar imágenes para rendimiento (tamaño y formato).
  • Usar srcset para imágenes responsivas.

2. Videos

Los videos se insertan usando la etiqueta <video>. Aquí están los atributos esenciales:

Ejemplo de Video

Atributos importantes:

  • src: Fuente del video (dentro de <source> para múltiples formatos).
  • controls: Muestra controles de reproducción.
  • autoplay: Reproduce el video automáticamente.
  • loop: Repite el video.
  • muted: Sin sonido por defecto.
  • poster: Imagen mostrada antes de reproducir el video.

Formatos recomendados:

  • MP4: Ampliamente soportado, ideal para la mayoría de los navegadores.
  • WebM: Alternativa moderna con buena compresión.

3. Iframes

El elemento <iframe> se usa para incrustar contenido externo, como videos de YouTube o mapas de Google.

Ejemplo de Iframe

Atributos importantes:

  • src: URL del contenido externo.
  • width y height: Tamaño del iframe.
  • frameborder: Controla el borde alrededor del iframe (valor 0 elimina el borde).
  • allowfullscreen: Permite ver el iframe en pantalla completa.

Consideraciones de seguridad:

  • Usar sandbox para limitar capacidades del contenido embebido y evitar problemas de seguridad.

4. Embebidos

La etiqueta <embed> se usa para insertar contenido multimedia o interactivo.

Ejemplo de Embebido

Atributos importantes:

  • src: Fuente del contenido.
  • width y height: Tamaño del área de embebido.

Uso recomendado:

  • Usar <embed> para documentos PDF y otros contenidos multimedia.

5. Figura y Leyendas

La etiqueta <figure> agrupa contenido multimedia y la etiqueta <figcaption> proporciona una leyenda o descripción.

Ejemplo de Figura con Leyenda

Uso de etiquetas:

  • <figure>: Agrupa la imagen o el video y su leyenda.
  • <figcaption>: Proporciona una descripción o créditos.

6. Buenas Prácticas en Multimedia

Consideraciones para la optimización y accesibilidad:

  • Optimización de rendimiento: Comprimir imágenes y videos para mejorar tiempos de carga.
  • Accesibilidad: Proporcionar alternativas como texto alternativo para imágenes y subtítulos para videos.
  • Carga diferida: Usar atributos como loading="lazy" para mejorar tiempos de carga en sitios con muchas imágenes.
