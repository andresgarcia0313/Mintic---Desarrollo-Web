Modelo de Caja en CSS

El modelo de caja define cómo los elementos HTML son representados como cajas rectangulares en la página. Cada caja tiene cuatro áreas:

  • Contenido: El área donde va el texto o imágenes.
  • Padding: Espacio entre el contenido y el borde.
  • Border: El borde que rodea el padding.
  • Margin: El espacio entre el borde y otros elementos.
